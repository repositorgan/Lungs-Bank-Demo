# Java Bank Account Project

## Overview
A simple Java console application that models a bank account.
This project demonstrates core Java concepts such as:
- Classes and objects
- Constructors
- Encapsulation
- Methods with parameters
- Conditional logic

## How It Works
1. A BankAccount object is created using a constructor.
2. The account starts with an initial balance.
3. Users can deposit and withdraw funds.
4. Account information is displayed to the console.

## How to Run
```bash
javac BankAccount.java
java BankAccount
