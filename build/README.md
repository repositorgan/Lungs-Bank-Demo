# Java Bank Account Project

## Overview
**A simple Java console application that models a bank account.**
**This project demonstrates core Java concepts such as:**
> Classes and objects

> Constructors

> Encapsulation

> Methods with parameters

> Conditional logic

## How It Works
> A BankAccount object is created using a constructor.

> The account starts with an initial balance.

>  Users can deposit and withdraw funds.

> Account information is displayed to the console.

## How to Run
```bash
javac BankAccount.java
java BankAccount
